# Tradutor Simultâneo de Propriedades

Ferramenta desenvolvida em Angular focada em produtividade. 

A ideia desse projeto nasceu da necessidade de otimizar o tempo no trabalho na hora de traduzir dezenas de propriedades estruturadas. Em vez de ficar traduzindo campo por campo manualmente, a ferramenta identifica a formatação (chaves e valores), extrai apenas os textos necessários, traduz simultaneamente para Inglês e Espanhol, e devolve a lista com a mesma formatação original do código.

## Como rodar localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor:
   ```bash
   npm start
   ```

Acesse em `http://localhost:4200/`.
